# Edu_LBSimulation: A Framework for Learning Behavioral Simulation with Generative Agents

This repository contains the partial implementation of the **Learning Behavioral Simulation (LBS)** framework introduced in our paper. The framework integrates **generative agents**, **multi-expert reflection**, and **interview-based memory construction** to simulate learner behaviors in educational settings.

This public release includes the core modules necessary to understand and examine the pipeline’s main components. The **full implementation**, including all experiment scripts and configuration files, will be released **upon acceptance of the manuscript**.

---

## 🔍 Overview

The LBS framework is designed to model and simulate learner behavior by combining:

- **Interview-based memory construction**  
  Structured interviews are converted into memory units via LLM-based processing.

- **Multi-expert reflective reasoning**  
  Multiple expert agents generate layered reflections to refine behavioral descriptors.

- **Retrieval-augmented behavioral generation**  
  Behavior simulation uses memory retrieval, reflection fusion, and LLM-mediated behavior synthesis.

The framework is domain-agnostic, allowing adaptation to various educational contexts, age groups, and cultural settings.
